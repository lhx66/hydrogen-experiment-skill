# 光纤氢气传感器实验自动化Skill

## 描述

自动化执行光纤氢气传感器实验，整合MFC控制、数据采集和数据分析功能。

## 使用前必读

**每次唤起实验时，必须询问用户实验结果存放在哪个文件夹中**

## 支持的实验请求

- "进行十次4%氢气测试，每次40秒，使用功率计测量"
- "进行5次2%氢气测试，每次30秒，使用FBG测量"
- "做三次1%氢气测试，每次20秒"

## 参数说明

| 参数 | 说明 | 默认值 |
|------|------|--------|
| 循环次数 | 实验重复次数 | 1 |
| 氢气浓度 | 通入氢气浓度 (%) | 用户指定 |
| 通氢时间 | 每次通氢持续时间 (秒) | 40 |
| 测量仪器 | 功率计 或 FBG解调仪 | 功率计 |
| 载气流量 | MFC2载气流量 (slm) | 2.0 |
| 循环间隔 | 每次循环间隔 (秒) | 60 |
| MFC串口 | MFC连接端口 | COM3 (波特率9600) |
| 输出文件夹 | **实验结果保存位置** | **必须询问用户** |

## 实验流程

1. 询问用户实验结果保存文件夹
2. 连接MFC和测量仪器
3. 对于每次循环：
   - 打开MFC1（通氢气）
   - 等待指定时间
   - 关闭MFC1
   - 等待数据采集完成
   - **分析数据并生成响应曲线图**
4. 关闭所有设备
5. **绘制所有循环的合并图并保存**
6. 生成实验报告

## 输出内容

### 实验目录结构

```
{用户指定的文件夹}/{timestamp}/
├── sensor1_4percent_cycle1_20240616_103046.csv
├── sensor1_4percent_cycle2_20240616_103546.csv
├── ...
├── sensor1_4percent_allcycles_20240616_110046.png  ← 所有循环合并图
└── experiment_results.json
```

### 每次循环后显示

- 响应曲线图（base64编码，在agent窗口显示）
- 分析结果：响应幅度、t90、信噪比等

### 实验完成后显示

- 所有循环的合并曲线图（保存到本地）
- 实验结果JSON文件

## 数据分析指标

| 指标 | 说明 |
|------|------|
| has_response | 是否检测到氢气响应 |
| response_amplitude | 响应幅度 |
| response_start_time | 响应起始时间 |
| t90 | 达到90%响应的时间 |
| recovery_time | 恢复到基线的时间 |
| signal_to_noise | 信噪比 |
| estimated_concentration_percent | 估算的氢气浓度 |

## 依赖工具

- mfc_cli.py - MFC控制工具（波特率9600）
- powermeter_cli.py - 功率计数据采集
- fbg_cli.py - FBG解调仪数据采集
- analyze_sensor_response.py - 数据分析模块（含绘图功能）

## 安全机制

- MFC2流量监测：当MFC2流量 < 0.1 slm时自动关闭MFC1
- 异常中断保护：Ctrl+C时优雅关闭所有设备
- 数据定期保存：每10个数据点flush到磁盘
